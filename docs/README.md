### FreezeYou
- 打开`FreezeYou`：`freezeyou://start`

- 冻结解冻指定应用：`freezeyou://fuf/?pkgName=[包名]`<sup>[引自](https://zidon.net/zh-CN/api/uri.html)</sup>
