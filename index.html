# FreezeYou

  • 打开FreezeYou：freezeyou://start
  • 打开FreezeYou：freezeyou://run
  • 冻结解冻指定应用：freezeyou://fuf/?pkgName=[包名]引自open in new window
  • 请求解冻指定应用：freezeyou://fuf/?pkgName=[包名]&action=unfreeze引自open in new window
