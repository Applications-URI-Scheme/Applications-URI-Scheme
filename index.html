                                              # FreezeYou

                                              • 打开FreezeYou：freezeyou://start
                                              • 打开FreezeYou：freezeyou://run
                                              • 冻结解冻指定应用：freezeyou://fuf/?pkgName=[包名]引自
                                              • 请求解冻指定应用：freezeyou://fuf/?pkgName=[包名]&action=unfreeze引自
                                              • 请求冻结指定应用：freezeyou://fuf/?pkgName=[包名]&action=freeze引自
                                              • 请求解冻并启动指定应用：freezeyou://fuf/?pkgName=[包名]&action=unFreezeAndRun引自
